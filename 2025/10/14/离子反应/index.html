离子反应

Posted on 2025-10-14 Views:

1. 概念

电解质

电解质：溶于水 或 熔融状态下可以导电的 化合物

  • 或：满足其一即可，如 \(\ce{HCl}\) 气体在熔融状态下无法导电，但水溶液可以；\(\ce{BaSO4}\) 溶于水几乎无法导电，但熔融状态下可以。

  • 注意是 化合物。另外，非电解质也要求是化合物。

  • 电解质导电是因为有自由移动的 离子，而单质导电是因为有自由移动的 电子

  • \(\ce{CO2, NH3}\) 不是电解质，因为有反应 \[ \ce{CO2 + H2O <=> H2CO3 \\ NH3 +H2O<=> NH3\cdot H2O} \] 自由移动的离子来源于 \(\ce{H2CO3, NH3\cdot H2O}\) 而不是 \(\ce{CO2,NH3}\)。

常见的电解质：酸碱盐、\(\ce{H2O}\)、活泼金属氧化物（熔融）

电解质强弱

电解质强弱：根据在水中的电离程度，完全电离为强，部分电离为弱。电解质的强弱和溶解性无关。

强电解质

  • 活泼金属氧化物：\(\ce{Na2O,CaO,BaO}\) 等
  • 强酸：\(\ce{H2SO4,HNO3,HClO4, HCl,HBr,HI}\)（高中化学六大强酸）等
  • 强碱：\(\ce{NaOH,KOH,Ba(OH)2,\color{red}Ca(OH)2}\) 等
  • 绝大多数盐（和溶解性无关）

弱电解质

  • 弱酸：\(\ce{CH3COOH, H2CO3, HF, HClO, H3BO3}\)（硼酸）
  • 弱碱：\(\ce{NH3\cdot H2O, Mg(OH)2, Fe(OH)3}\)
  • 极少数盐：\(\ce{(CH3COOH)2Pb, Fe(SCN)3}\)（硫氰化铁）
  • 水（大约每 \(10^7\) 个水分子中会有 \(1\) 个发生电离 \(\ce{H2O <=> H+ + OH-}\)）

电离方程式

电离：电解质溶于水 或 受热熔化，形成自由移动离子的过程（物理变化，条件是溶解或熔化，无需通电）

  • 以 \(\ce{NaCl}\) 为例，在水中电离出水合钠离子和水合氯离子，而熔融态只电离出钠离子和氯离子。
  • 电离示意图中，要体现异性相吸，如水合钠离子，应由 \(\ce{O^2-}\) 朝向钠离子

电解：通直流电，并在电极处发生氧化还原反应（化学变化，需通直流电）

电离方程式书写

  • 强电解质电离用 \(\ce{=}\)，弱电解质电离用 \(\ce{<=>}\) \[ \begin{aligned} \ce{ H2SO4 = 2H+ + SO4^2- \\ H2CO3 <=> 2H+ + CO3^2- \\ Ba(OH)2 = Ba^2+ + 2OH- \\ CH3COOH <=> CH3COO- + H+ \\ NH3\cdot H2O <=> NH4+ + OH- } \end{aligned} \]

  • 绝大多数多元弱酸酸式根（如 \(\ce{HCO3-, HSO3-}\)）在水中不能完全电离，但是 \(\ce{HSO4-}\) 在水中可以 \[ \ce{ NaHSO4 = Na+ + H+ + SO4^2- } \]

​ 但在熔融状态下也不行 \[ \ce{ NaHSO4 = Na+ + HSO4- } \]

​ 这是因为溶解电离破坏的是离子键（水合作用）和共价键，而熔融态只能破坏离子键不能破坏共价键。

  • 多元弱酸分步电离，以第一步为主。 \[ \begin{aligned} \ce{ H2CO3 <=> H+ + HCO3- \\ HCO3- <=> H+ + CO3^2- } \end{aligned} \]

  • 多元弱碱分步电离，视为全部电离，用 \(\ce{<=>}\) 连接。 \[ \ce{Al(OH)3 <=> Al^3+ + 3OH-} \]

2. 离子反应

离子方程式书写

离子方程式书写

  • 拆：易溶于水且易电离的物质（易溶 + 强电解质）（强酸、强碱、大部分可溶性盐）写成离子形式。

  • 在水溶液中完全以自由移动的离子形式存在的物质才能拆成离子

  • 浓硫酸不能拆，体系中的水太少了。\(\ce{H2SO4}\) 分子依赖水电离，浓硫酸中大量 \(\ce{H2SO4}\) 分子未电离，不拆。

  • 强碱 \(\ce{Ca(OH)2}\) 是微溶物，微溶物作为产物一律不拆，作为反应物，澄清则拆，浑浊不拆。

    • 高中四大微溶物：\(\ce{Ca(OH)2, MgCO3, CaSO4, AgSO4}\)

  • 易溶的强电解质盐，若作为产物以沉淀形式析出，不拆。如侯氏制碱法： \[ \begin{aligned} \ce{ NaCl + H2O + NH3 = NH4Cl + NaHCO3 v \\ Na+ + H2O + NH3 = NH4+ + NaHCO3 v } \end{aligned} \]

  • 氧化物不拆

离子反应发生条件

复分解型 离子反应发生的条件（大原则：离子减少）

  • 溶解度大制溶解度小

    • \(\ce{CaCl2}\) 溶液和 \(\ce{NaOH}\) 溶液反应 \[ \ce{ Ca2+ + 2OH- = Ca(OH)2 v } \]

  • 难挥发（沸点高）制易挥发（沸点低）

    • 稀硫酸与 \(\ce{Na2CO3}\) 溶液反应 \[ \ce{ 2H+ + CO3^2- = CO2 ^ + H2O } \]

  • 强电解质制弱电解质

    • 醋酸钠与盐酸反应 \[ \ce{ CH3COO- + H+ = CH3COOH } \]

  • 上述三条件满足其一即可，部分反应会违背规律，如 \[ \ce{ H2S + CuSO4 = CuS v + H2SO4 } \] 虽然违反了强酸制弱酸，但满足溶解度大制溶解度小

3. 离子共存

有色离子

常见有色离子：

颜色 橙 黄 绿 蓝 紫红
离子 \(\ce{Cr2O7^2-}\)（重铬酸根） \(\ce{Fe^3+}\) \(\ce{Fe^2+, Cr^3+}\) \(\ce{Cu^2+}\) \(\ce{MnO4-}\)

注意，溶液一定是澄清透明的，但不一定无色，有色和澄清透明不冲突。另外溶液一定呈电中性，虽然插在这里不太合适。

离子共存

常见反应组合（无法共存）

首先 \(\ce{H+, OH-}\) 无法大量共存不必说

1. 因生成弱电解质而无法大量共存

离子 不能大量共存的离子
\(\ce{H+}\) 1. 弱酸阴离子 \(\ce{CO3^2-, HCO3-,SO3^2-, HSO3-,S^2-, HS-, PO4^3-, H2PO4-,HPO4^2-,ClO-,F-}\) 等与 \(\ce{H+}\) 反应生成弱酸 2. \(\ce{[Al(OH)4]-}\) 与 \(\ce{H+}\) 反应生成 \(\ce{Al(OH)3}\) 或 \(\ce{Al^3+}\)
\(\ce{OH-}\) 1. 弱碱阳离子 \(\ce{Mg^2+, Al^3+, Fe^3+, Fe^2+, Cu^2+, NH4+}\) 等与 \(\ce{OH-}\) 产生弱碱而不能大量共存 2. 弱酸的酸式酸根 \(\ce{HCO3-, HSO3-, HS-, H2PO4-, HPO4^2-}\) 等与 \(\ce{OH-}\) 反应生成水和酸根离子 3. \(\ce{Ca^2+}\) 和 \(\ce{OH-}\) 无法大量共存

2. 因生成沉淀（或微溶物）而无法大量共存

离子 不能大量共存的离子
\(\ce{Cl-,Br-,I-}\) \(\ce{Ag+}\)
\(\ce{SO4^2-}\) \(\ce{Ba^2+,Pb^2+, Ca^2+, Ag+}\)
\(\ce{CO3^2-,SO3^2-,PO4^3-,S^2-}\) 除 \(\ce{Na+,K+,NH4+}\) 外的大多数阳离子
\(\ce{OH-}\) 除 \(\ce{Na+,K+,Ba^2+}\) 外的大多数阳离子

注：\(\ce{PbSO4}\) 难溶；\(\ce{AgCl}\) 白，\(\ce{AgBr}\) 淡黄，\(\ce{AgI}\) 黄

离子检验

  • \(\ce{H+}\)：氢前金属、酸碱指示剂、\(\ce{pH}\) 试纸
  • \(\ce{OH-}\)：酸碱指示剂、\(\ce{pH}\) 试纸
  • \(\ce{SO4^2-}\)：过量盐酸酸化，无明显现象；加入 \(\ce{BaCl2}\) 溶液，产生白色沉淀
  • \(\ce{Cl-}\)：加入硝酸酸化的 \(\ce{AgNO3}\) 溶液，产生白色沉淀
  • \(\ce{CO3^2-}\)：加入过量 \(\ce{BaCl2}\) 溶液（区别 \(\ce{HCO3-}\)），过滤并洗涤，将沉淀加入盐酸，沉淀溶解并生成 无色无味（区别 \(\ce{SO2-}\)）且能使澄清石灰水变浑浊的气体

4. 电解质与导电性

能导电的物质不一定是电解质（\(\ce{Fe,C, NaCl}\) 水溶液），电解质不一定能导电（既不充分也不必要）

溶于水能导电的物质不一定是电解质（\(\ce{CO2,SO2,NH3}\)）

电解质强弱与导电性强弱没有必然关系

导电性强弱取决于离子浓度与电荷量

5. 与量有关的离子方程式书写（重要）

技巧：

  • 少量定为 \(1\)，过量要多少有多少
  • 先中和，后沉淀
  • 少量符合化学式比例，过量不一定
  • 先难溶，后微溶

\(\ce{NaHCO3}\) 溶液和 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{NaHCO3}\) 少量 \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

  • \(\ce{Ca(OH)2}\) 少量 \[ \ce{ Ca^2+ + 2OH- + 2HCO3- = CaCO3 v + CO3^2- + 2H2O } \]

\(\ce{Ca(HCO3)2}\) 溶液与 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{Ca(HCO3)2}\) 少量： \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

  • \(\ce{Ca(OH)2}\) 少量： \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

\(\ce{CO2}\) 通入 \(\ce{NaOH}\) 溶液中：

  • \(\ce{CO2}\) 少量： \[ \ce{ CO2 + 2OH- = CO3^2- + 2H2O } \]

  • \(\ce{CO2}\) 过量： \[ \ce{ OH- + CO2 = HCO3- } \]

  • 少量 \(\ce{CO2}\) 生成 \(\ce{CO3^2-}\)，过量 \(\ce{CO2}\) 生成 \(\ce{HCO3-}\)

\(\ce{CO2}\) 通入澄清石灰水中：

  • \(\ce{CO2}\) 少量： \[ \ce{ CO2 + Ca^2+ + 2OH- = CaCO3 v + 2H2O } \]

  • \(\ce{CO2}\) 过量（由于 \(\ce{Ca(HCO3)2}\) 可溶，\(\ce{Ca^2+}\) 不参与反应） \[ \ce{ OH- + CO2 = HCO3- } \]

\(\ce{NH4HSO4}\) 溶液和 \(\ce{NaOH}\) 溶液反应：

  • \(\ce{NaOH}\) 少量： \[ \ce{ OH- + H+ = H2O } \]

  • \(\ce{NaOH}\) 过量： \[ \ce{ NH4+ + H+ + 2OH- = NH3\cdot H2O + H2O } \]

\(\ce{Mg(HCO3)2}\) 溶液和 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{Ca(OH)2}\) 少量： \[ \ce{ Ca^2+ + 2OH- + Mg^2+ + 2HCO3- = CaCO3 v + 2H2O + MgCO3 v } \]

  • \(\ce{Ca(OH)2}\) 过量： \[ \ce{ Mg^2+ + 2HCO3- + 4OH- + 2Ca^2+ = Mg(OH)2 v + 2H2O + 2CaCO3 v } \]
