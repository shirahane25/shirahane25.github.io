0. 化合价

恒定的化合价（优先判断）

  • 单质化合价 \(0\)

  • \(\ce{IA(Li, Na,K, Rb,Cs)}\) 一定 \(+1\)，\(\ce{IIA(Mg, Ca, Sr, Ba)}\) 一定 \(+2\)，\(\ce{Al}\) 一定 \(+3\)，\(\ce{Zn}\) 一定 \(+2\)，\(\ce{Ag}\) 一定 \(+1\)

  • \(\ce{F}\) 得电子能力最强，所以一定 \(-1\) 价，比氧还强，所以在 \(\ce{OF2}\) 中，\(\ce{O}\) 是 \(+2\) 价。

  • \(\ce{H}\) 元素 + 非金属 一般 \(+1\) 价，+ 金属 \(-1\) 价。但是 \(\ce{SiH4}\) 和 \(\ce{B2H6}\) 中 \(\ce{H}\) 是\(-1\) 价。

  • \(\ce{O}\) 一般 \(-2\) 价，\(-1\) 价的是过氧化物。\(\ce{KO2}\) 超氧化钾，\(\ce{O}\) 为 \(-\dfrac{1}{2}\) 价，还有氟氧化物 \(\ce{OF2}\) 和 \(\ce{O2F2}\)。

非金属元素族的化合价区间

  • \(\ce{IVA(C,Si)}\) 最高化合价 \(+4\)，最低化合价 \(-4\)
  • \(\ce{VA(N,P)}\) 最高化合价 \(+5\)，最低化合价 \(-3\)
  • \(\ce{VIA(S)}\) 最高化合价 \(+6\)，最低化合价 \(-2\)
  • \(\ce{VIIA(Cl,Br,I)}\) 最高化合价 \(+7\)，最低化合价 \(-1\)

过渡金属

高中需要知道的过渡金属 \[ \huge\ce{\underset{钪}{Sc},\underset{钛}{Ti},\underset{钒}{V},\underset{铬}{Cr},\underset{锰}{Mn},\underset{铁}{Fe},\underset{钴}{Co},\underset{镍}{Ni},\underset{铜}{Cu},\underset{锌}{Zn},\underset{银}{Ag},\underset{金}{Au}} \]

铁

  • \(\ce{FeO,Fe2O3}\) 氧化亚铁氧化铁无需多言
  • \(\ce{Fe3O4}\) 可以看作 \(\ce{FeO\cdot Fe2O3}\)，铁的化合价为 \(+2, +3, +3\)
  • \(\ce{K2FeO4}\) 高铁酸钾中 \(\ce{Fe}\) 是 \(+6\) 价
  • \(\ce{K3[Fe(CN)6]}\) 六氰合铁酸钾，或铁氰化钾。\(\ce{Fe}\) 为 \(+3\) 价（\(\ce{CN-}\)：\(-1\) 价，\(\ce{KCN}\) 氰化钾（氰化物）剧毒）
  • \(\ce{FeS2}\) 硫化亚铁（黄铁矿），\(\ce{Fe}\) \(+2\) \(\ce{S}\) \(-1\)
  • \(\ce{CuFeS2}\) 只知道 \(\ce{S}\) 是 \(-2\)，\(\ce{Cu,Fe}\) 是 \(+2,+2\) 还是 \(+1, +3\) 存在争议

氨配合物

  • \(\ce{[Ag(NH3)2]OH}\)：\(\ce{[Ag(NH3)2]+}\) 二氨合银离子，\(\ce{Ag}\) \(+1\) 价，\(\ce{NH3}\) 是中性分子整体 \(0\) 价。
  • \(\ce{[Cu(NH3)4]SO4}\)，\(\ce{Cu}\) \(+2\) 价

常考过渡金属

下面四个过渡金属最高化合价直接背下来（尤其是 \(\ce{Cr, Mn}\)）

  • \(\ce{Ti}\) 钛，最高化合价 \(+4\)，\(\ce{TiO2}\) 钛白粉，用于涂改液。
  • \(\ce{V}\) 钒，最高化合价 \(+5\)，\(\ce{V2O5}\)
  • \(\ce{Cr}\) 铬，最高化合价 \(+6\)，\(\ce{CrO4^2-}\) 铬酸根（黄），\(\ce{Cr2O7^2-}\) 重铬酸根（强氧化剂）。\(\ce{K2Cr2O7}\) 重铬酸钾
  • \(\ce{Mn}\) 锰，最高化合价 \(+7\)，\(\ce{KMnO4}\) 氧化性超强

1. 概念

氧化还原反应

氧化还原反应：有化合价变化（电子转移）的化学反应。

本质：电子转移（电子得失或共用电子对偏移）

特征：反应过程中元素化合价变化

化合价与电子转移

\[ \ce{ 2Na + Cl2 \overset{\Delta}{=} 2NaCl } \]

  • \(\ce{Na}\) 失去 \(\ce{2e^-}\)，化合价升高，被氧化
  • \(\ce{Cl}\) 得到了 \(\ce{2e-}\)，化合价降低，被还原
  • 物质失电子则化合价升高
  • 物质得电子则化合价降低

\[ \ce{ H2 + Cl2 \overset{点燃}{=} 2HCl } \]

  • \(\ce{H}\) 化合价升高，被氧化
  • \(\ce{Cl}\) 化合价降低，被还原
  • 共用电子对偏离自己的原子（原子核吸引电子的能力弱一些）则化合价升高
  • 共用电子对偏向自己的原子（原子核吸引电子的能力强一些）则化合价降低

氧化还原反应与四大基本反应的关系

化合反应与氧化还原反应有交不包含（黑色 在交集中） \[ \begin{aligned} \ce{ 2Na + Cl2 &\overset{\Delta}{=} 2NaCl \\ \color{red}NH3 + HCl &= NH4Cl } \end{aligned} \] 分解反应与氧化还原反应有交但不包含 \[ \begin{aligned} \ce{ 2H2O2 &\xlongequal{\ce{MnO2}} 2H2O \\ 2KMnO4 &\xlongequal{\Delta} K2MnO4 + MnO2 + O2 ^ \\ \color{red} CaCO3 &\xlongequal{高温} CaO + CO2 ^ \\ 2NaHCO3 &\xlongequal{\Delta}Na2CO3 + CO2 ^ + H2O } \end{aligned} \] 一般 有单质参加 / 生成的化合 / 分解反应是氧化还原反应，反例：同素异形体的转化 \(\ce{O2 <=> O3}\)

置换反应 包含于 氧化还原反应

复分解反应与氧化还原反应 无交

离子反应主要包括复分解型和氧化还原型

氧化还原反应的分类

  • 歧化反应：同种物质、同种元素的同一价态，反应中化合价既升高又降低 \[ \begin{aligned} \ce{ Cl2 + 2NaOH &\xlongequal{} NaCl + NaClO + H2O \\ 2NaO2 + 2H2O &= 4NaOH + O2 ^ \\ 3S + 6NaOH &= 2Na2S + Na2SO3 + 3H2O } \end{aligned} \]

  • 归中反应：同一元素的 高低两种化合价，反应后化合价向中间价态靠拢但不交叉 \[ \begin{aligned} \ce{ 2H2S + SO2 &= 3S v + 2H2O \\ KClO3 + 6HCl &= 3Cl2 ^ + KCl + 3H2O \\ 2FeCl3 + Fe &= 3FeCl2 } \end{aligned} \]

  • 部分氧化还原反应：变价元素的原子只有部分化合价改变 \[ \begin{aligned} \ce{ 8NH3 + 3Cl2 &= 6NH4Cl + N2 \\ 3Cu + 8HNO3(稀) &= 3Cu(NO3)2 + 2NO ^ + 4H2O } \end{aligned} \]

  • 完全氧化还原反应：变价元素的原子全部参加反应 \[ \begin{aligned}\ce{ 4NH3 + 5O2 &\xlongequal[\Delta]{催化剂} 4NO + 6H2O \\ 4FeS2 + 11O2 &\xlongequal{\Delta}2Fe2O3 + 8SO2 } \end{aligned} \] \(\ce{FeS2}\)：二硫化亚铁（硫化合价 \(-1\)）

判断氧化还原反应时，\(\ce{O,H}\) 的优先级较低。

氧化剂、还原剂

氧化剂

氧化剂：反应中得到电子（所含元素化合价降低）的反应物

常见氧化剂（化合价容易降低的物质）

  • 非金属单质：\(\ce{O2, Cl2, Br2}\)
  • 含有高价态元素的化合物：\(\ce{HNO3,KMnO4,MnO2, 浓H2SO4, K2Cr2O7}\)
  • 某些金属阳离子：\(\ce{Fe^3+,Ag+,Cu^2+}\)
  • 过氧化物：\(\ce{H2O2,Na2O2,CaO2}\)
  • \(\ce{HClO,ClO-,HNO2}\) 等

还原剂

还原剂：反应中失去电子（所含元素化合价升高）的反应物

常见还原剂（化合价容易升高的物质）

  • 活泼金属单质：\(\ce{K,Ca,Na,Mg,Al,Zn,Fe}\)

  • 非金属元素的简单阴离子及低价态化合物：\(\ce{S^2-, I-, CO,SO2,H2SO3,Na2SO3}\)

  • 某些低价态金属阳离子：\(\ce{Fe^2+, Cu+}\)

  • 非金属单质及氢化物：\(\ce{H2, C,H2S,NH3}\)

  • 氢元素化合价为 \(-1\) 的化合物：\(\ce{LiH,NaBH4(四氢合硼钠), NaH}\) 等

3. 新情境氧化还原反应方程式书写

写几道题吧

草酸（\(\ce{H2C2O4}\)） 与硫酸酸化的高锰酸钾反应，生成 \(\ce{MnSO4}\) 和 \(\ce{CO2}\)

\[ \ce{ 5H2C2O4 + 2KMnO4 + 3H2SO4 = 2MnSO4 + 10CO2 ^ + K2SO4 + 8H2O } \]

将 \(\ce{ClO2}\) 通入 \(\ce{KI}\)、淀粉溶液、稀盐酸的混合溶液中，\(\ce{ClO2}\) 被还原成 \(\ce{Cl-}\) 且溶液变蓝

\[ \ce{ 2ClO2 + 10KI + 8HCl = 10KCl + 5I2 + 4H2O } \]

为什么 \(\ce{I2}\) 不用标沉淀符号 \(\ce{v}\) 呢？

\(\ce{I2}\) 要说明一下，正常情况下碘单质的溶解度极低（\(25^{\circ}\rm C\) 时 \(0.03g / 100g \ce{H2O}\)），纯碘水溶液颜色很浅，也不稳定，与淀粉几乎不嫌蓝，所以淀粉实验中，总是用 \(\ce{KI}\) 溶液帮助碘溶解，配成 碘酒。将碘单质加入 \(\ce{KI}\) 溶液中： \[ \ce{ I2 + I- <=> I3- } \] \(\ce{I3-}\)（三碘离子）是一种稳定、易溶于水的络合离子，碘酒实际上是三碘化钾溶液，准确名称 \(\ce{KI - I2}\) 混合溶液。

因此淀粉实验中的碘酒是含有 \(\ce{I2}\) 和 过量 \(\ce{I-}\) 的溶液，主要成分 \(\ce{I3-}\)。

而淀粉显色的原理是，淀粉与 \(\ce{I3-}\) 形成的包络络合物 \[ \ce{ starch + I3- -> [starch\cdot I3-](bluecomplex) } \] 淀粉分子里的螺旋结构恰好能容纳 \(\ce{I3-}\) 链，光吸收后表现为 深蓝色。

所以，单独的 \(\ce{I2}\) 是没办法使淀粉变蓝的。

回到最开始的问题，溶液变蓝意味着溶液中存在 \(\ce{I3-}\)，意味着 \(\ce{I-}\) 是过量的，\(\ce{I2}\) 不会沉淀析出。

\(\ce{NiSO4}\) 溶液在强碱溶液中用 \(\ce{NaClO}\) 氧化，可沉淀出用作镍铬电池正极材料的 \(\ce{NiOOH}\)，已知 \(\ce{ClO-}\) 被还原为 \(\ce{Cl-}\)，写出该反应的离子方程式

\[ \ce{ 2Ni^2+ + ClO- + 4OH- = 2NiOOH v + Cl- + H2O } \]

用 \(\ce{NaClO3, Na2SO3}\) 固体和浓硫酸制取 \(\ce{ClO2}\) 气体。已知 \(\ce{Na2SO3}\) 被氧化成 \(\ce{Na2SO4}\)

\[ \ce{ 2NaClO3 + Na2SO3 + H2SO4 = 2Na2SO4 + 2ClO2 ^ + H2O } \]

\(\ce{NaOH,H2O2}\) 混合溶液中通入 \(\ce{ClO2}\) 气体制取 \(\ce{NaClO2}\) 的离子方程式

\[ \ce{ 2ClO2 + H2O2 + 2OH- = 2ClO2- + O2 + 2H2O } \]

4. 氧化还原反应的强弱规律

强强联手

\[ \huge\ce{ \underset{较强氧化性}{氧化剂} + \underset{较强还原性}{还原剂} = \underset{弱还原性}{还原产物} + \underset{弱氧化性}{氧化产物} } \]

例题：

常温下，溶液中可以发生以下反应：

  • \(\ce{2Fe^2+ + Br2 = 2Fe^3+ + 2Br-}\)（氧化性：\(\ce{Br2 > Fe^3+}\)）
  • \(\ce{2Br- + Cl2 = Br2 + 2Cl-}\)（氧化性：\(\ce{Cl2 > Br2 }\)）
  • \(\ce{2Fe^3+ + 2I- = 2Fe^2+ + I2}\)（氧化性：\(\ce{Fe^3+ > I2}\)）

综上得出，氧化性：\(\ce{Cl2 > Br2 > Fe^3+ > I2}\)

对应的还原产物的还原性就会相反：\(\ce{Cl- < Br- < Fe^2+ < I-}\)

强者先行

同一氧化剂，与含多种还原剂（浓度相同）的溶液反应时，首先被氧化的是 还原性最强 的物质。

同一还原剂，与含多种氧化剂（浓度相同）的溶液反应时，首先被还原的是 氧化性最强 的物质。

还原性顺序（被氧化的顺序）

\[ \huge\ce{ \overset{-2}{S} > \overset{+4}{S} > I- > Fe^2+ > Br- > Cl- > Mn^2+ } \]

口诀：留点铁修路，猛哥。

例题：

将少量氯水滴入 \(\ce{FeBr2}\) 溶液，发生的反应是 \[ \ce{ 2Fe^2+ + Cl2 = 2Fe^3+ + 2Cl- } \] 将足量氯水滴入 \(\ce{FeBr2}\) 溶液，发生的反应是 \[ \ce{2Fe^2+ + 4Br- + 3Cl2 = 2Fe^3+ + 2Br2 + 6Cl-} \] 将少量氯水滴入 \(\ce{FeI2}\) 溶液，发生的反应是 \[ \ce{ 2I- + Cl2 = I2 + 2Cl- } \] 将足量氯水滴入 \(\ce{FeI2}\) 溶液，发生的反应是 \[ \ce{ 2Fe^2+ + 4I- + 3Cl2 = 2Fe^3+ + 2I2 + 6Cl- } \]

金属、单质的还原性强弱判断

金属活动性顺序表中，越往右，金属越不活泼，金属单质的还原性逐渐减弱，金属阳离子的氧化性逐渐增强 \[ \large\ce{ \xrightarrow[K+ Ca^2+ Na+ Mg^2+ Al^3+ Zn^2+ Fe^3+ Sn^2+ Pb^2+ (H+) Cu^2+ Hg^2+ Ag+ ]{K \ Ca \ Na \ Mg \ Al \ Zn \ Fe \ Sn \ Pb \ (H) \ Cu \ Hg \ Ag} } \] 单质氧化性：\(\ce{F2 > Cl2 > Br2 > I2}\)

阴离子还原性：\(\ce{F- < Cl- < Br- < I-}\)

实战

判断题

  • 得电子越多的氧化剂，其氧化性就越强（\(\times\)）

  • 失电子越多的还原剂，其还原性就越强（\(\times\)）

  • 同一元素化合价越高，氧化性一定越强（\(\times\)）重要反例：氧化性：\(\ce{H\overset{\color{red}+7}{Cl}O4 < H\overset{\color{blue}+1}{Cl}O}\)

  • 含有最高价元素的化合物一定具有很强的氧化性（\(\times\)）一举一堆反例：\(\ce{CO2, Al2O3}\)

某溶液含有 \(\ce{Cl-, I-}\)。为了氧化 \(\ce{I-}\) 而不氧化 \(\ce{Cl-}\)，可选用的氧化剂是？ \[ \rm A. \ce{FeCl2} \ \ \ \ \ \ \ \ B. \ce{KMnO4} \ \ \ \ \ \ \ \ C. 浓盐酸 \ \ \ \ \ \ \ \ \color{red}D.\ce{FeCl3} \] 分析：氧化 \(\ce{I-, Cl-}\) 对应的氧化产物是 \(\ce{I2, Cl2}\)，所以该氧化剂的氧化性要强于 \(\ce{I2}\)，弱于 \(\ce{Cl2}\)，根据还原性顺序表 \(\ce{I- > Fe^2+ > Cl-}\) 推出氧化性顺序 \(\ce{I2 < Fe^3+ < Cl2}\)，故选用 \(\rm D. \ce{FeCl3}\)。

5. 陌生氧化还原反应方程式书写

在稀硫酸中，重铬酸钾氧化过氧化氢

\[ \ce{ K2Cr2O7 + 3H2O2 + 4H2SO4 = K2SO4 + Cr2(SO4)3 + 3O2 ^ + 7H2O } \]

\(\ce{NO}\) 在碱性条件下被 \(\ce{H2O2}\) 氧化成 \(\ce{NO3-}\)

\[ \ce{ 2NO + 3H2O2 + 2OH- = 2NO3- + 4H2O } \]

\(\ce{HNO2}\) 与酸性 \(\ce{(NH2)2CO}\) 水溶液生成 \(\ce{N2}\) 和 \(\ce{CO2}\)

\[ \ce{ 2HNO2 + (NH2)2CO = 2N2 ^ + CO2 ^ + 3H2O } \]

将砷废渣（主要成分 \(\ce{As2S3}\)）制成浆状通入 \(\ce{O2}\) 氧化，生成 \(\ce{H3AsO4}\) 和单质硫

\[ \ce{ 2As2S3 + 5O2 + 6H2O = 4H3AsO4 + 6S } \]

酸性次氯酸钠溶液中，次氯酸氧化一氧化氮生成氯离子和硝酸根离子

\[ \ce{ 3HClO + 2NO + H2O = 3Cl- + 2NO3- + 5H+ } \]

制备次氯酸钠溶液，若温度超过 \(40 ^{\circ}\rm C\)，氯气与氢氧化钠溶液反应生成氯酸钠和氯化钠

\[ \ce{ 3Cl2 + 6OH- \xlongequal{\Delta} ClO3- + 5Cl- + 3H2O } \]

次氯酸钠溶液与银单质反应生成氯化银、氢氧化钠、氧气

\[ \ce{ 4NaClO + 4Ag + 2H2O = 4AgCl + 4NaOH + O2 } \]

次氯酸钠溶液与氨制备联氨（\(\ce{N2H4}\)，又称肼，无色液体）

\[ \ce{ NaClO + 2NH3 = N2H4 + NaCl + H2O } \]

磷酸铁、碳酸锂、草酸高温煅烧制取磷酸亚铁锂（\(\ce{LiFePO4}\)）

\[ \ce{ 2FePO4 + Li2CO3 + H2C2O4 \xlongequal{高温} 2LiFePO4 + 3CO2 ^ + H2O } \]

硫酸、二氧化硫、次氯酸钠制取二氧化氯并回收硫酸氢钠

\[ \ce{ H2SO4 + SO2 + 2NaClO3 = 2ClO2 + 2NaHSO4 } \]

1. 概念

电解质

电解质：溶于水 或 熔融状态下可以导电的 化合物

  • 或：满足其一即可，如 \(\ce{HCl}\) 气体在熔融状态下无法导电，但水溶液可以；\(\ce{BaSO4}\) 溶于水几乎无法导电，但熔融状态下可以。

  • 注意是 化合物。另外，非电解质也要求是化合物。

  • 电解质导电是因为有自由移动的 离子，而单质导电是因为有自由移动的 电子

  • \(\ce{CO2, NH3}\) 不是电解质，因为有反应 \[ \ce{CO2 + H2O <=> H2CO3 \\ NH3 +H2O<=> NH3\cdot H2O} \] 自由移动的离子来源于 \(\ce{H2CO3, NH3\cdot H2O}\) 而不是 \(\ce{CO2,NH3}\)。

常见的电解质：酸碱盐、\(\ce{H2O}\)、活泼金属氧化物（熔融）

电解质强弱

电解质强弱：根据在水中的电离程度，完全电离为强，部分电离为弱。电解质的强弱和溶解性无关。

强电解质

  • 活泼金属氧化物：\(\ce{Na2O,CaO,BaO}\) 等
  • 强酸：\(\ce{H2SO4,HNO3,HClO4, HCl,HBr,HI}\)（高中化学六大强酸）等
  • 强碱：\(\ce{NaOH,KOH,Ba(OH)2,\color{red}Ca(OH)2}\) 等
  • 绝大多数盐（和溶解性无关）

弱电解质

  • 弱酸：\(\ce{CH3COOH, H2CO3, HF, HClO, H3BO3}\)（硼酸）
  • 弱碱：\(\ce{NH3\cdot H2O, Mg(OH)2, Fe(OH)3}\)
  • 极少数盐：\(\ce{(CH3COOH)2Pb, Fe(SCN)3}\)（硫氰化铁）
  • 水（大约每 \(10^7\) 个水分子中会有 \(1\) 个发生电离 \(\ce{H2O <=> H+ + OH-}\)）

电离方程式

电离：电解质溶于水 或 受热熔化，形成自由移动离子的过程（物理变化，条件是溶解或熔化，无需通电）

  • 以 \(\ce{NaCl}\) 为例，在水中电离出水合钠离子和水合氯离子，而熔融态只电离出钠离子和氯离子。
  • 电离示意图中，要体现异性相吸，如水合钠离子，应由 \(\ce{O^2-}\) 朝向钠离子

电解：通直流电，并在电极处发生氧化还原反应（化学变化，需通直流电）

电离方程式书写

  • 强电解质电离用 \(\ce{=}\)，弱电解质电离用 \(\ce{<=>}\) \[ \begin{aligned} \ce{ H2SO4 = 2H+ + SO4^2- \\ H2CO3 <=> 2H+ + CO3^2- \\ Ba(OH)2 = Ba^2+ + 2OH- \\ CH3COOH <=> CH3COO- + H+ \\ NH3\cdot H2O <=> NH4+ + OH- } \end{aligned} \]

  • 绝大多数多元弱酸酸式根（如 \(\ce{HCO3-, HSO3-}\)）在水中不能完全电离，但是 \(\ce{HSO4-}\) 在水中可以 \[ \ce{ NaHSO4 = Na+ + H+ + SO4^2- } \]

​ 但在熔融状态下也不行 \[ \ce{ NaHSO4 = Na+ + HSO4- } \]

​ 这是因为溶解电离破坏的是离子键（水合作用）和共价键，而熔融态只能破坏离子键不能破坏共价键。

  • 多元弱酸分步电离，以第一步为主。 \[ \begin{aligned} \ce{ H2CO3 <=> H+ + HCO3- \\ HCO3- <=> H+ + CO3^2- } \end{aligned} \]

  • 多元弱碱分步电离，视为全部电离，用 \(\ce{<=>}\) 连接。 \[ \ce{Al(OH)3 <=> Al^3+ + 3OH-} \]

2. 离子反应

离子方程式书写

离子方程式书写

  • 拆：易溶于水且易电离的物质（易溶 + 强电解质）（强酸、强碱、大部分可溶性盐）写成离子形式。

  • 在水溶液中完全以自由移动的离子形式存在的物质才能拆成离子

  • 浓硫酸不能拆，体系中的水太少了。\(\ce{H2SO4}\) 分子依赖水电离，浓硫酸中大量 \(\ce{H2SO4}\) 分子未电离，不拆。

  • 强碱 \(\ce{Ca(OH)2}\) 是微溶物，微溶物作为产物一律不拆，作为反应物，澄清则拆，浑浊不拆。

    • 高中四大微溶物：\(\ce{Ca(OH)2, MgCO3, CaSO4, AgSO4}\)

  • 易溶的强电解质盐，若作为产物以沉淀形式析出，不拆。如侯氏制碱法： \[ \begin{aligned} \ce{ NaCl + H2O + NH3 = NH4Cl + NaHCO3 v \\ Na+ + H2O + NH3 = NH4+ + NaHCO3 v } \end{aligned} \]

  • 氧化物不拆

离子反应发生条件

复分解型 离子反应发生的条件（大原则：离子减少）

  • 溶解度大制溶解度小

    • \(\ce{CaCl2}\) 溶液和 \(\ce{NaOH}\) 溶液反应 \[ \ce{ Ca2+ + 2OH- = Ca(OH)2 v } \]

  • 难挥发（沸点高）制易挥发（沸点低）

    • 稀硫酸与 \(\ce{Na2CO3}\) 溶液反应 \[ \ce{ 2H+ + CO3^2- = CO2 ^ + H2O } \]

  • 强电解质制弱电解质

    • 醋酸钠与盐酸反应 \[ \ce{ CH3COO- + H+ = CH3COOH } \]

  • 上述三条件满足其一即可，部分反应会违背规律，如 \[ \ce{ H2S + CuSO4 = CuS v + H2SO4 } \] 虽然违反了强酸制弱酸，但满足溶解度大制溶解度小

离子检验

  • \(\ce{H+}\)：氢前金属、酸碱指示剂、\(\ce{pH}\) 试纸
  • \(\ce{OH-}\)：酸碱指示剂、\(\ce{pH}\) 试纸
  • \(\ce{SO4^2-}\)：过量盐酸酸化，无明显现象；加入 \(\ce{BaCl2}\) 溶液，产生白色沉淀
  • \(\ce{Cl-}\)：加入硝酸酸化的 \(\ce{AgNO3}\) 溶液，产生白色沉淀
  • \(\ce{CO3^2-}\)：加入过量 \(\ce{BaCl2}\) 溶液（区别 \(\ce{HCO3-}\)），过滤并洗涤，将沉淀加入盐酸，沉淀溶解并生成 无色无味（区别 \(\ce{SO2-}\)）且能使澄清石灰水变浑浊的气体

4. 电解质与导电性

能导电的物质不一定是电解质（\(\ce{Fe,C, NaCl}\) 水溶液），电解质不一定能导电（既不充分也不必要）

溶于水能导电的物质不一定是电解质（\(\ce{CO2,SO2,NH3}\)）

电解质强弱与导电性强弱没有必然关系

导电性强弱取决于离子浓度与电荷量

5. 与量有关的离子方程式书写（重要）

技巧：

  • 少量定为 \(1\)，过量要多少有多少
  • 先中和，后沉淀
  • 少量符合化学式比例，过量不一定
  • 先难溶，后微溶

\(\ce{NaHCO3}\) 溶液和 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{NaHCO3}\) 少量 \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

  • \(\ce{Ca(OH)2}\) 少量 \[ \ce{ Ca^2+ + 2OH- + 2HCO3- = CaCO3 v + CO3^2- + 2H2O } \]

\(\ce{Ca(HCO3)2}\) 溶液与 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{Ca(HCO3)2}\) 少量： \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

  • \(\ce{Ca(OH)2}\) 少量： \[ \ce{ Ca^2+ + HCO3- + OH- = CaCO3 v + H2O } \]

\(\ce{CO2}\) 通入 \(\ce{NaOH}\) 溶液中：

  • \(\ce{CO2}\) 少量： \[ \ce{ CO2 + 2OH- = CO3^2- + 2H2O } \]

  • \(\ce{CO2}\) 过量： \[ \ce{ OH- + CO2 = HCO3- } \]

  • 少量 \(\ce{CO2}\) 生成 \(\ce{CO3^2-}\)，过量 \(\ce{CO2}\) 生成 \(\ce{HCO3-}\)

\(\ce{CO2}\) 通入澄清石灰水中：

  • \(\ce{CO2}\) 少量： \[ \ce{ CO2 + Ca^2+ + 2OH- = CaCO3 v + 2H2O } \]

  • \(\ce{CO2}\) 过量（由于 \(\ce{Ca(HCO3)2}\) 可溶，\(\ce{Ca^2+}\) 不参与反应） \[ \ce{ OH- + CO2 = HCO3- } \]

\(\ce{NH4HSO4}\) 溶液和 \(\ce{NaOH}\) 溶液反应：

  • \(\ce{NaOH}\) 少量： \[ \ce{ OH- + H+ = H2O } \]

  • \(\ce{NaOH}\) 过量： \[ \ce{ NH4+ + H+ + 2OH- = NH3\cdot H2O + H2O } \]

\(\ce{Mg(HCO3)2}\) 溶液和 \(\ce{Ca(OH)2}\) 溶液反应：

  • \(\ce{Ca(OH)2}\) 少量： \[ \ce{ Ca^2+ + 2OH- + Mg^2+ + 2HCO3- = CaCO3 v + 2H2O + MgCO3 v } \]

  • \(\ce{Ca(OH)2}\) 过量： \[ \ce{ Mg^2+ + 2HCO3- + 4OH- + 2Ca^2+ = Mg(OH)2 v + 2H2O + 2CaCO3 v } \]
